# ANGULAR INVENTORY APP

A Simple Inventory App In Angular

## Acknowledgements

- [Ng Book](https://www.amazon.com/ng-book-Complete-Angular-Nathan-Murray/dp/1985170280)

## Deployment

To deploy this project run

```bash
  npm install -g now
  cd dist/inventory-app
  now
```
